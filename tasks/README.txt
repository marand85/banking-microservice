As part of this task, create an application that listens on port 8080 and handles the HTTP protocol.

The application should expose endpoints solving three problems located in the atmservice, onlinegame and transactions directories.
Endpoint specifications in the OpenAPI 3.0 standard are in the .json files.
Example requests and responses are contained in the example_request.json and example_response.json files.

Good luck and have fun! :)
